Privacy Policy for DocuSign CLM Favicon Switcher

Effective Date: July 7, 2025

Overview

DocuSign CLM Favicon Switcher (“the Extension”) is a browser add-on whose sole purpose is to read the aid query parameter from DocuSign CLM pages and inject the corresponding environment-specific favicon. It does not collect, store, or transmit any personal or usage data.

Data Collection

  • What We Read: Only the current page URL’s aid parameter, in memory.
  • What We Don’t Collect: No user identifiers, no browsing history, no form data, no analytics.

Data Storage & Transmission

  • All processing is done in memory on the client side.
  • No data is written to disk, sent to any server, or logged.

Third-Party Access

  • We do not share, sell, or transfer any data to third parties.
  • This Extension does not include any remote scripts or analytics libraries.

Security

  • All code is bundled locally within the Extension package.
  • There are no network requests for code or data beyond loading the page itself.

Changes to This Policy

If this policy is updated, the “Effective Date” above will be revised. Users will be notified via the Chrome Web Store update notes.
